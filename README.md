# VandaSystem

Sistema de gestão de vendas e produtos.

## Funcionalidades

- Cadastro de produtos
- Controle de vendas
- Relatórios financeiros
- Configurações do sistema

## Tecnologias

- HTML5
- CSS3 
- JavaScript
- LocalStorage

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Cadastre produtos
3. Realize vendas
4. Acompanhe relatórios
